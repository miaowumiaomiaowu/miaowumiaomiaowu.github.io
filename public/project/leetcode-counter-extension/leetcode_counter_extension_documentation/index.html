LeetCode 刷题统计扩展

项目概述

LeetCode 刷题统计扩展 是一个简洁高效的浏览器扩展，专为开发者设计，用于追踪和统计在 LeetCode 上的刷题记录。通过直观的可视化数据展示，帮助用户构建持续学习的编程习惯，提升算法能力。

  • 开发者: 韩嘉仪 (miaowumiaomiaowu)
  • 技术栈: JavaScript + HTML + CSS
  • 项目类型: 浏览器扩展
  • 许可证: MIT License
  • GitHub: miaowumiaomiaowu/leetcode-counter-extention

核心特性

📊 智能数据统计

实时刷题追踪

  • 自动识别: 无需手动操作，自动检测 LeetCode 题目完成状态
  • 实时更新: 即时记录每道题目的解题时间和完成状态
  • 多维度统计: 按难度、标签、时间等维度进行数据分析

详细进度记录

  • 解题历史: 完整记录每次刷题的时间戳和题目信息
  • 完成率统计: 分类统计简单、中等、困难题目的完成情况
  • 连续刷题天数: 追踪学习连续性，激励持续学习

📈 可视化数据展示

直观图表分析

  • 进度条显示: 清晰展示各难度题目的完成进度
  • 趋势图表: 展示刷题频率和学习趋势变化
  • 热力图: 可视化每日刷题活跃度分布

个性化面板

  • 简洁界面: 优雅的弹窗设计，不干扰正常刷题流程
  • 快速预览: 一键查看关键统计数据和学习成果
  • 自定义设置: 根据个人喜好调整显示内容和样式

🎯 学习助手功能

智能提醒系统

  • 学习目标: 设置每日/每周刷题目标，保持学习节奏
  • 温和提醒: 非侵入式提醒，鼓励持续学习
  • 成就系统: 解锁各种学习里程碑，增强成就感

数据导出与分享

  • 数据备份: 支持学习数据的导出和备份
  • 进度分享: 生成精美的学习报告图片，分享学习成果
  • 跨设备同步: 通过云同步保持多设备数据一致

技术架构

前端技术栈

核心技术:
- Vanilla JavaScript (73.6%) - 核心逻辑实现
- CSS3 (18.8%) - 样式设计和动画效果
- HTML5 (7.6%) - 页面结构和扩展界面

浏览器扩展技术:
- Chrome Extensions API
- Content Scripts 内容脚本
- Background Scripts 后台脚本
- Storage API 本地存储
- Popup 弹窗界面

扩展架构设计

Manifest 配置

{
  "manifest_version": 3,
  "name": "LeetCode 刷题统计",
  "version": "1.0.0",
  "permissions": [
    "storage",
    "activeTab"
  ],
  "content_scripts": [
    {
      "matches": ["*://leetcode.com/*", "*://leetcode.cn/*"],
      "js": ["content.js"]
    }
  ],
  "action": {
    "default_popup": "popup.html"
  }
}

核心模块

  • Content Script: 页面内容监听和数据采集
  • Background Script: 后台数据处理和状态管理
  • Popup Interface: 用户交互界面和数据展示
  • Storage Manager: 本地数据存储和管理

数据处理流程

  1. 页面监听: Content Script 监听 LeetCode 页面变化
  2. 状态识别: 自动识别题目完成状态和相关信息
  3. 数据存储: 将解题记录存储到浏览器本地存储
  4. 实时更新: 更新统计数据和可视化图表
  5. 用户展示: 通过 Popup 界面展示统计结果

功能模块详解

🔍 智能页面监听

  • DOM 变化检测: 实时监听页面元素变化
  • 题目状态识别: 自动识别题目是否已完成
  • 多语言支持: 同时支持 LeetCode 国际版和中文版

📊 数据统计引擎

  • 多维度分析: 按难度、标签、时间等维度统计
  • 趋势计算: 计算学习趋势和进步速度
  • 目标跟踪: 追踪个人学习目标完成情况

🎨 用户界面设计

  • 响应式设计: 适配不同屏幕尺寸和分辨率
  • 暗色模式: 支持明暗主题切换
  • 动画效果: 流畅的过渡动画和交互反馈

安装与使用

安装步骤

# 1. 克隆项目到本地
git clone https://github.com/miaowumiaomiaowu/leetcode-counter-extention.git

# 2. 在 Chrome 浏览器中打开扩展管理页面
chrome://extensions/

# 3. 开启开发者模式

# 4. 点击"加载已解压的扩展程序"，选择项目文件夹

使用指南

  1. 访问 LeetCode: 在 LeetCode 网站正常刷题
  2. 自动统计: 扩展会自动记录您的刷题进度
  3. 查看统计: 点击浏览器扩展图标查看详细统计
  4. 设置目标: 在设置页面配置个人学习目标

项目亮点

💡 创新特性

  1. 零干扰设计: 完全不影响正常的 LeetCode 使用体验
  2. 智能识别: 无需手动标记，自动识别题目完成状态
  3. 可视化友好: 直观的图表展示，让数据说话

  • 透明开源: 代码完全开源，用户可自主审查

🔧 开发优势

  • 模块化设计: 清晰的代码结构，易于维护和扩展
  • 兼容性强: 支持主流 Chromium 内核浏览器
  • 性能优化: 高效的数据处理和 DOM 操作

技术实现细节

Content Script 核心逻辑

// 监听页面变化，自动检测题目完成状态
class LeetCodeTracker {
  constructor() {
    this.initializeTracker();
  }
  
  initializeTracker() {
    // 初始化页面监听器
    this.observePageChanges();
    this.detectProblemCompletion();
  }
  
  observePageChanges() {
    // 使用 MutationObserver 监听 DOM 变化
    const observer = new MutationObserver(() => {
      this.checkProblemStatus();
    });
    observer.observe(document.body, { childList: true, subtree: true });
  }
}

开发理念

LeetCode 刷题统计扩展 的开发遵循以下核心理念：

  1. 用户体验优先: 零干扰设计，完全融入原生 LeetCode 体验
  2. 数据驱动学习: 通过可视化数据激励持续学习
  3. 隐私保护: 用户数据完全本地化，保护个人隐私
  4. 简洁高效: 功能专注，性能优化，轻量级设计
  5. 开放透明: 开源代码，社区驱动，持续改进

性能优化

  • 懒加载: 按需加载功能模块，减少初始化时间
  • 防抖处理: 优化页面监听频率，避免过度计算
  • 缓存机制: 智能缓存计算结果，提升响应速度
  • 异步处理: 非阻塞式数据处理，保持界面流畅

未来规划

  • 多平台支持: 扩展到 Firefox、Safari 等其他浏览器
  • 数据分析增强: 更深入的学习模式分析和建议
  • 社交功能: 学习进度分享和好友对比功能
  • AI 推荐: 基于刷题历史推荐适合的题目
  • 云同步: 可选的云端数据同步功能
  • 移动端支持: 开发手机端配套应用
